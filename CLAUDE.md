# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is an Obsidian vault for English language learning. The owner is a Russian-speaking IT programmer (backend/general dev) targeting employment in the UK or USA. Current level: ~B1–B2 receptive (reads English docs comfortably), weaker productive skills (speaking/writing). Primary goal: bridge the gap to professional working English.

## Vault structure

```
Grammar/
  Tenses/          ← core grammar notes, start here
  Sentences/       ← word order, questions, negations
  Modal Verbs/     ← can/could/would/should/must
Vocabulary/        ← to be created later
Templates/
  Grammar Template.md   ← use for new grammar notes
```

## Note conventions

- All notes are bilingual: Russian explanation + English examples.
- Notes use Obsidian wikilinks: `[[Present Simple]]`, `[[Overview]]`.
- Frontmatter fields: `type`, `topic`, `level` (beginner/intermediate/advanced), `reviewed`.
- IT-context sections show how grammar appears in real work: standups, PRs, Slack, interviews.
- Grammar notes follow the pattern: Formula → When to use → Examples table → IT context → Common mistakes → See also.

## Content priorities

Grammar tenses order (highest priority first): Present Simple → Present Continuous → Past Simple → Present Perfect → Future (will/going to) → Past Continuous → Conditionals.

When creating new grammar content, always include an IT-work section with realistic phrases a programmer would use in standups, code reviews, or job interviews.
