# Времена английского языка / Tenses Overview

Приоритет для IT-работы за рубежом:

| Приоритет | Время | Зачем нужно |
|-----------|-------|-------------|
| ⭐⭐⭐ | [[Present Simple]] | Факты, документация, инструкции |
| ⭐⭐⭐ | [[Present Continuous]] | "Я сейчас делаю...", текущие задачи |
| ⭐⭐⭐ | [[Past Simple]] | Что сделал, что произошло |
| ⭐⭐⭐ | [[Present Perfect]] | "Я сделал..." (результат важен сейчас) |
| ⭐⭐ | [[Future: will / going to]] | Планы, договорённости |
| ⭐⭐ | [[Past Continuous]] | Фон для событий прошлого |
| ⭐ | [[Past Perfect]] | "Уже сделал к тому моменту" |
| ⭐ | [[Conditionals]] | "Если бы...", "Если сделаешь..." |

## Ключевое правило

Английские времена передают не только **когда**, но и **как** — завершено действие или нет, важен результат или процесс. Это главное отличие от русского языка.
