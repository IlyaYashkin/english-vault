# Future: will / going to

В английском нет отдельного "будущего времени" как формы глагола. Будущее выражают несколько способов, главные два — **will** и **going to**.

## Структура / Formula

### will

```
+ I / He / She / We / ...   will (I'll / he'll)     work
- I / He / She / We / ...   will not (won't)         work
?                            Will   I / he / she ...  work?
```

### going to

```
+ I           am going to    work
+ He/She/It   is going to    work
+ We/You/They are going to   work

- I'm not going to / He isn't going to / ...
? Are you going to work? / Is he going to work?
```

## Когда использовать / When to use

| | **will** | **going to** |
|---|---|---|
| **Решение** | Принято прямо сейчас, спонтанно | Принято заранее, есть план |
| **Пример** | *I'll take that task.* | *I'm going to refactor the module this sprint.* |
| **Предсказание** | Мнение, без доказательств | Видно по ситуации, есть признаки |
| **Пример** | *I think it'll take 2 days.* | *The build is going to fail — there are errors.* |
| **Обещание / предложение** | ✅ | ❌ |
| **Пример** | *I'll help you with that.* | — |

> Практически: **going to** = есть план до разговора. **will** = решение или реакция в момент разговора.

## Примеры / Examples

| Русский | English |
|---------|---------|
| Я возьму этот таск. (решил сейчас) | I'll take that task. |
| На этой неделе я планирую написать тесты. | I'm going to write tests this week. |
| Думаю, это займёт день. | I think it'll take a day. |
| Смотри, там опечатка — билд упадёт. | Look, there's a typo — the build is going to fail. |
| Помогу тебе с ревью. | I'll help you with the review. |
| Мы не будем делать рефакторинг в этом спринте. | We're not going to refactor it this sprint. |

## IT-контекст / In IT work

**Спонтанные решения (will):**
- *I'll assign that to myself.*
- *I'll look into it after the call.*
- *Don't worry, I'll fix it.*

**Планы на спринт / итерацию (going to):**
- *I'm going to work on the API integration this week.*
- *We're going to migrate the database next sprint.*
- *Are you going to add tests for that?*

**Оценки времени (will):**
- *It'll probably take 3–4 hours.*
- *I don't think it'll be ready today.*
- *This won't break anything — it's just a config change.*

**На интервью:**
- *Where do you see yourself in 3 years?* → *I'll probably move into a tech lead role.*

## Present Continuous как будущее

Для **конкретно запланированного события** (встреча, созвон, деплой с договорённостью) часто используют [[Present Continuous]]:

- *I'm meeting the client tomorrow at 3pm.*
- *We're deploying on Friday.*

Это звучит более конкретно и уверенно, чем *going to*.

## Частые ошибки / Common mistakes

- ❌ *I will to go* → ✅ *I will go* (после will — базовая форма)
- ❌ *I will go to refactor* → ✅ *I'm going to refactor* (есть план)
- ❌ *He going to fix it* → ✅ *He **is** going to fix it*
- ❌ *I'll not do it* → ✅ *I **won't** do it*
