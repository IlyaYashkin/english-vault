# Past Simple

## Структура / Formula

```
+ I / He / She / It / We / You / They   worked      (правильные глаголы: + -ed)
+ I / He / She / It / We / You / They   went        (неправильные: 2-я форма)

- I / ... did not (didn't) work

? Did I / he / she / ... work?
```

> После **did / didn't** глагол всегда в базовой форме: *Did he go?* (не ~~went~~)

### Правильные глаголы: как добавить -ed

| Окончание | Правило | Пример |
|---|---|---|
| -e на конце | + -d | fix**e**d → fixed |
| согласная + -y | -y → -ied | deploy → deployed |
| короткий гласный + согласный | удвоить согласный | commit → committed |
| остальные | + -ed | work → worked |

### Неправильные глаголы — запомнить

| Base | Past Simple | Перевод |
|------|-------------|---------|
| go | went | идти / ехать |
| have | had | иметь |
| make | made | делать |
| find | found | находить |
| write | wrote | писать |
| break | broke | ломать |
| build | built | строить |
| run | ran | запускать |
| read | read | читать (произн. [red]) |
| send | sent | отправлять |
| meet | met | встречаться |
| get | got | получать |
| come | came | приходить |
| say | said | говорить |
| think | thought | думать |

## Когда использовать / When to use

1. **Завершённое действие в прошлом** — есть конкретное время или ясно из контекста
2. **Последовательность событий** в прошлом
3. **Привычки и состояния в прошлом**, которых уже нет

Маркеры: *yesterday, last week/month/year, ago, in 2023, on Monday*

## Примеры / Examples

| Русский | English |
|---------|---------|
| Я исправил баг вчера. | I fixed the bug yesterday. |
| Мы задеплоили фичу в пятницу. | We deployed the feature on Friday. |
| Сервер упал час назад. | The server went down an hour ago. |
| Ты проверил логи? | Did you check the logs? |
| Я не успел дописать тесты. | I didn't finish writing the tests. |

## IT-контекст / In IT work

> Past Simple — язык пост-мортемов, ретро и отчётов о проделанной работе.

**На ретро / в отчёте:**
- *We **released** version 2.1 last sprint.*
- *I **implemented** the OAuth flow.*
- *The pipeline **failed** because of a missing env variable.*
- *We **migrated** the database last Thursday.*

**На стендапе (вчера):**
- *Yesterday I **finished** the PR and **sent** it for review.*
- *I **found** a bug in the auth module.*

## Частые ошибки / Common mistakes

- ❌ *I go to the office yesterday* → ✅ *I **went***
- ❌ *Did he **went**?* → ✅ *Did he **go**?*
- ❌ *I didn't **fixed** it* → ✅ *I didn't **fix** it*
- ❌ *We have deployed* (это [[Present Perfect]]) vs *We deployed last Friday* (Past Simple — есть конкретное время)
