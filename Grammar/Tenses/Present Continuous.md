# Present Continuous

## Структура / Formula

```
+ I           am (I'm)       working
+ He/She/It   is (he's)      working
+ We/You/They are (we're)    working

- I           am not (I'm not)        working
- He/She/It   is not (isn't)          working
- We/You/They are not (aren't)        working

? Am   I              working?
? Is   he / she / it  working?
? Are  we / you / they working?
```

### Как добавить -ing

| Окончание | Правило | Пример |
|---|---|---|
| -e на конце | убрать -e, добавить -ing | make → making |
| короткий гласный + согласный | удвоить согласный | run → running, sit → sitting |
| остальные | + -ing | work → working |

## Когда использовать / When to use

1. **Прямо сейчас** — действие происходит в момент разговора
2. **Временный период** — сейчас, но не навсегда
3. **Запланированное будущее** (с датой или контекстом)
4. **Изменяющаяся ситуация** (с: getting, becoming, changing)

## Примеры / Examples

| Русский | English |
|---------|---------|
| Я сейчас разбираю баг. | I'm investigating a bug right now. |
| Мы сейчас мигрируем на новую БД. | We're migrating to a new database. |
| Он сейчас на созвоне. | He's on a call right now. |
| Ты деплоишь сейчас? | Are you deploying right now? |
| Я встречаюсь с командой завтра. | I'm meeting the team tomorrow. |

## IT-контекст / In IT work

> Present Continuous — это язык стендапа: что ты делаешь сейчас и что планируешь.

**На стендапе:**
- *I'm **working on** the login feature.*
- *I'm **fixing** the issue with the API timeout.*
- *I'm **reviewing** John's pull request.*
- *We're **moving** our CI/CD to GitHub Actions.*

**Планы (ближайшее будущее):**
- *I'm **deploying** the hotfix this afternoon.*
- *We're **releasing** v2.0 on Friday.*

## Глаголы, которые НЕ используются в Continuous

Эти глаголы описывают состояния, а не действия — у них нет "процесса":

| Категория | Глаголы |
|---|---|
| Мышление | know, understand, believe, think (мнение), remember, forget |
| Эмоции | like, love, hate, want, need, prefer |
| Чувства | see, hear, smell, taste |
| Владение | have (иметь), own, belong |

- ❌ *I am knowing the answer* → ✅ *I know the answer*
- ❌ *He is having a car* → ✅ *He has a car*

Исключение: *I'm having lunch* (have = есть/принимать пищу — действие, не владение)

## Частые ошибки / Common mistakes

- ❌ *I working* → ✅ *I **am** working*
- ❌ *He is work* → ✅ *He is work**ing***
- ❌ *I am understanding* → ✅ *I understand*
