# Present Perfect

## Структура / Formula

```
+ I / You / We / They   have (I've / we've)   worked
+ He / She / It         has  (he's / she's)   worked

- I / You / We / They   have not (haven't)   worked
- He / She / It         has not  (hasn't)    worked

? Have  I / you / we / they   worked?
? Has   he / she / it         worked?
```

**3-я форма глагола (Past Participle):**
- Правильные: те же что Past Simple → *work → worked, fix → fixed*
- Неправильные: отдельная форма → *go → gone, write → written, build → built*

## Когда использовать / When to use

1. **Результат важен сейчас** — действие в прошлом, но эффект в настоящем
2. **Опыт** — было ли когда-либо (ever, never)
3. **Недавнее прошлое** (just, recently, lately)
4. **Незавершённый период** (today, this week, this year — период ещё не закончился)

> Ключевое: **конкретное время НЕ называется**. Как только появляется *yesterday / last week / in 2023* — это [[Past Simple]].

## Примеры / Examples

| Русский | English |
|---------|---------|
| Я сделал это (и это важно сейчас). | I've done it. |
| Мы исправили баг. (результат важен) | We've fixed the bug. |
| Я никогда не работал с Rust. | I've never worked with Rust. |
| Ты деплоил это раньше? | Have you ever deployed this? |
| Я только что отправил PR. | I've just sent the PR. |
| Мы выпустили три версии на этой неделе. | We've released three versions this week. |

## IT-контекст / In IT work

> Present Perfect — это язык статусов и обновлений: "что уже готово".

**На стендапе (результат):**
- *I'**ve** finished the feature — it's ready for review.*
- *I'**ve** found the root cause of the issue.*
- *We'**ve** merged the PR.*

**В Slack / комментариях:**
- *I'**ve** pushed the fix.*
- *I'**ve** updated the documentation.*
- *Has anyone **seen** this error before?*

**Опыт в интервью:**
- *I'**ve** worked with microservices for 3 years.*
- *I'**ve** never used Kubernetes in production.*
- *Have you **ever** led a team?*

## Present Perfect vs Past Simple — главная путаница

| Ситуация | Present Perfect | Past Simple |
|---|---|---|
| Конкретное время? | Нет | Да |
| Пример | *I've fixed it.* | *I fixed it yesterday.* |
| Период закончился? | Нет (this week) | Да (last week) |
| Пример | *I've worked hard this week.* | *I worked hard last week.* |

## Частые ошибки / Common mistakes

- ❌ *I have fixed it yesterday* → ✅ *I **fixed** it yesterday* (есть конкретное время)
- ❌ *I have never went* → ✅ *I have never **gone***
- ❌ *He have done* → ✅ *He **has** done*
- ❌ *Did you ever try?* → ✅ *Have you ever tried?*
